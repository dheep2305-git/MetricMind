# Semantic Layer

## Overview

The semantic layer acts as the bridge between the database and the AI model.

It defines business metrics and provides a structured representation of business data.

## Technologies

- Cube.dev
- SQL
- Snowflake

## Responsibilities

- Define business metrics.
- Create dimensions and measures.
- Manage data models.
- Improve query performance.
