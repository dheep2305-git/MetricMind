# Frontend

## Overview

The frontend is responsible for providing the user interface for the MetricMind platform.

Users will be able to ask business-related questions in natural language and view AI-generated insights through an interactive dashboard.

## Technologies

- Next.js
- React.js
- JavaScript
- HTML
- CSS

## Features

- User-friendly dashboard
- Natural language query input
- Interactive charts and graphs
- Business KPI cards
- AI-generated insights
- Responsive design
