# LLM

## Overview

The LLM (Large Language Model) module enables users to interact with the system using natural language.

It converts user questions into business queries and generates meaningful insights from the available data.

## Technologies

- LangChain
- OpenAI LLM
- Python

## Responsibilities

- Process natural language queries.
- Generate SQL queries.
- Communicate with Cube.dev.
- Return AI-generated responses.
