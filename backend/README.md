# Backend

## Overview

This folder contains the FastAPI backend for the MetricMind project.

The backend acts as the central component that connects the frontend, LLM, Cube.dev, and Snowflake database.

## Technologies

- Python
- FastAPI
- Snowflake
- Cube.dev
- LangChain
- REST API

## Current Status

- Initial backend structure created.
- Basic API routes added.
- FastAPI application initialized.
